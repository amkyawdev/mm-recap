06 / Data Flow

Architecture & Data Flow

User Actions Flow

1. Upload Video
User → Next.js → Appwrite Storage
                    ↓
            2. Process Video
            video-processor (FFmpeg)
                    ↓
            3. Generate Subtitles
            subtitle-generator (Whisper)
                    ↓
            4. AI Auto-Edit
            ai-orchestrator (GPT-4/Gemini)
                    ↓
            5. Publish
            social-publisher (YouTube/TikTok)

API Architecture

BFF Pattern — Next.js API Routes act as Backend-for-Frontend, providing:

  • API key protection (not exposed to client)
  • Request/response transformation
  • Caching layer
  • Rate limiting

Authentication

  • ✓ Appwrite Auth for user management
  • ✓ API Keys stored in user settings
  • ✓ OAuth flows for YouTube/TikTok
